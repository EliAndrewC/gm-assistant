# Adventure Hooks Reference

Pre-written adventure hooks organized by setting type. These can be used as inspiration for generating new hooks or as starting points for campaign sessions.

## Files

- [countryside.md](countryside.md) - Hooks for rural village/hamlet settings: irrigation disputes, bandit sightings, strange occurrences in farmland
- [town.md](town.md) - Hooks for county town settings: market day incidents, magistrate problems, merchant disputes
- [grifts.md](grifts.md) - Scams and confidence tricks that PCs might encounter or investigate
- [city.md](city.md) - Hooks for provincial cities and domain capitals: riots, corruption, political intrigue
- [caravan.md](caravan.md) - Hooks for travel and caravan escort scenarios
- [prison.md](prison.md) - Hooks for prison camp settings
